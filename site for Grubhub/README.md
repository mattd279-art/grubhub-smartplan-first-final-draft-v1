# Grubhub SmartPlan Prototype

This is a self-contained, interactive UI prototype built with plain HTML/CSS/JS.

## Run locally

Option A (quick): open `index.html` in your browser.

Option B (recommended): start a simple server:

```bash
cd "site"
python3 -m http.server 8000
```

Then open: `http://localhost:8000`

## What’s included

- Onboarding / preferences (calories, budget, cooking frequency, dietary filters)
- Weekly Plan Dashboard (Cook vs Order toggles + calories)
- Automatically generated Grocery List (from cooked meals)
- Checkout AI Assistant (fee/discount transparency + contextual message)
- Budget Intelligence signals (budget progress + warnings)
- Grubhub+ Optimization (selectable deals)
- Grubhub+ Family Plan (multiple profiles + shared credits)
- Meal details modal (ingredients + step-by-step instructions for cooked meals)
